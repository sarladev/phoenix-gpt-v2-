# Phoenix GPT v2

ChatGPT-style AI assistant website using HTML, CSS, JavaScript, and OpenRouter API.

## Features

- ChatGPT-style UI
- Thinking animation
- Typing effect
- Voice input
- Voice output
- Chat history storage
- Dark/Light theme
- Mobile responsive design
- File upload support for text files
- Copy response
- Regenerate response
- Clear chat
- Download chat
- OpenRouter-compatible AI integration

## Setup

1. Create OpenRouter API key.
2. Open website.
3. Click API Key button.
4. Paste your key.
5. Start chatting.

## Deploy

GitHub Repository Settings > Pages > Deploy from Branch > main > root
